## 📊 How It Works

### Example: "Add JWT Authentication"

1. **Queen** establishes strategic plan
2. **Collective Intelligence** builds consensus on approach
3. **Memory Manager** sets up coordination namespace
4. **Workers** execute in parallel:
   - Backend worker → API endpoints
   - Database worker → User schema
   - Test worker → Unit & integration tests
   - Security scout → Vulnerability scan
5. **Reviewer** validates code quality
6. **Queen** reports completion

All coordination happens through **distributed memory** (`swarm/*` namespace) - no manual coordination needed!
